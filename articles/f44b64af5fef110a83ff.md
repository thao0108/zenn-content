## はじめに
Javascriptでpaizaの入力値を配列に分解する方法をメモします。

## メモ
例
入力値
1 2
3 4
5 6

```js
// 取得した入力値を配列に分解
var inputs = [];
lines.forEach(value => {
    result = value.split(/,|\s/);
    inputs.push(result)
})

// 出力
// [
//   [ '1', '2' ],
//   [ '3', '4' ],
//   [ '5', '6' ],
// ]  
```

inputsの中の文字列を全て数値にしたい時

```js
inputs = inputs.map(input => (
    input.map(Number)
));

// 出力
// [
//     [1, 2],
//     [3, 4],
//     [5, 6],
// ]

```
